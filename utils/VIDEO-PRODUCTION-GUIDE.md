## üé¨ Option 1: Using Narakeet (Recommended)

### Step 1: Prepare Your Slides

Create 44 slides (1920x1080 PNG images) named:
- `slide-1.png`
- `slide-2.png`
- `slide-3.png`
- ... through `slide-44.png`

**Design Tips:**
- Use BrainMark brand colors
- Include screenshots of the extension
- Minimal text on each slide
- High-quality visuals
- Consistent styling

### Step 2: Upload to Narakeet

1. Go to https://www.narakeet.com/
2. Create a new project
3. Upload your slide images
4. Copy content from `narakeet-script.md`
5. Paste into Narakeet's script editor

### Step 3: Configure Voice Settings

**Recommended Settings:**
```
Voice: Amy (en-US) - Professional female
Speed: 1.0 (natural pace)
Pitch: 1.0 (normal)
Style: Professional/Conversational
```

**Alternative Voices:**
- **Brian (en-GB)** - British male, authoritative
- **Joanna (en-US)** - Warm American female
- **Matthew (en-US)** - Clear American male

### Step 4: Add Background Music

Upload or select:
- Corporate/Tech upbeat music
- Volume: 20-30% of narration
- Fade in/out: 2-3 seconds
- No lyrics

### Step 5: Generate Video

1. Review pronunciation of technical terms
2. Adjust timing if needed
3. Click "Generate Video"
4. Wait 5-10 minutes
5. Download MP4 file

**Expected Output:**
- Duration: ~4 minutes 30 seconds
- Resolution: 1920x1080 (Full HD)
- Format: MP4
- Size: ~200-300 MB

## üé• Option 2: Professional Voice-Over Recording

### Using `narration-only.txt`:

1. **Hire Voice Talent:**
   - Fiverr: $50-150 for professional VO
   - Upwork: $100-300 for experienced talent
   - Voices.com: Premium quality

2. **Recording Specs:**
   - Format: WAV or MP3 (320kbps)
   - Sample Rate: 48kHz
   - Bit Depth: 24-bit
   - Mono or Stereo

3. **Direction for Voice Artist:**
   ```
   Style: Professional yet friendly
   Pace: 140 words per minute
   Tone: Informative, engaging
   Emphasis: Technical terms, benefits
   Pauses: Marked in script
   ```

4. **Video Editing:**
   - Use Adobe Premiere Pro or DaVinci Resolve
   - Sync voice-over with screen recordings
   - Add transitions, animations
   - Background music
   - Text overlays for key points
